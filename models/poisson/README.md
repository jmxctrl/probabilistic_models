## Poisson Distribution

### Definition
Let \( X \sim \text{Poisson}(\lambda) \) with pmf:
\[
P(X = k) = e^{-\lambda} \frac{\lambda^k}{k!}
\]

### Properties
- Mean: \( \lambda \)
- Variance: \( \lambda \)
- Additivity

### Interpretation
Counts events in a fixed interval under independence assumptions.

### Connections
- Limit of Binomial
- Poisson process increments
